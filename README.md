# 🚀 ESA OPS-SAT Mission Control Telemetry Dashboard

An end-to-end analytics project using **Dataiku DSS**, **Power BI**, and **DAX** to monitor satellite telemetry, detect anomalous patterns, and support predictive maintenance using real ESA OPS-SAT mission data.

---

# 📖 Overview

This project analyzes real telemetry data collected from the European Space Agency's (ESA) OPS-SAT mission to monitor spacecraft health, detect anomalous telemetry patterns, and support predictive maintenance.

The project follows an end-to-end analytics workflow, combining **Dataiku DSS** for data preparation and feature engineering with **Power BI** for interactive dashboards and operational decision support.

The objective is to transform raw satellite telemetry into meaningful insights that help Mission Control engineers monitor spacecraft performance and identify abnormal behaviours before they become critical issues.

# 📖 Business Story

Mission Control engineers receive thousands of telemetry measurements from ESA's OPS-SAT satellite every day.

The purpose of this dashboard is to provide operational visibility into spacecraft health by transforming complex telemetry data into intuitive business and engineering insights.

The dashboard is designed to answer three key operational questions:

1. Is the satellite operating normally?
2. Which telemetry channels require immediate attention?
3. Can telemetry anomalies help anticipate future failures?
